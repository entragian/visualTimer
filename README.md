# visualTimer

Offline PWA for building and running chain timers for rehab sessions.

## V1 scope

- Session editor with blocks and steps
- Repeat count per block
- Local persistence
- Static PWA shell
- Flat preview of the execution chain

## Build order

1. Lock down the data model for sessions, blocks, and steps.
2. Finish visual editing for blocks and steps.
3. Add playback state machine for the timer player.
4. Add sounds, vibration, and completion behavior.
5. Polish the PWA install/offline path.

## Run

Open `index.html` in a browser, or serve the folder with any static web server.
